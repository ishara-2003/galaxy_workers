WhatsApp Business API තොරතුරු

WhatsApp Business ID, Phone ID, සහ Access Token ලබාගන්නේ කෙසේද?

සිංහල: WhatsApp Business API භාවිතා කිරීමට, ඔබට Meta Developer ගිණුමක් අවශ්‍යයි. පහත පියවර අනුගමනය කරන්න:

  1. Meta Developer වෙබ් අඩවියට යන්න.
  2. Sign Up කර Meta Business Manager ගිණුමක් සාදන්න.
  3. Create App > Business තෝරන්න > App නම: Galaxy Workers.
  4. WhatsApp > API Setup > Cloud API තෝරන්න > Phone number එකතු කරන්න (+94).
  5. Configuration වලින් Business ID, Phone ID, සහ Access Token copy කරගන්න.

English: To use WhatsApp Business API, you need a Meta Developer account. Follow these steps:

  1. Visit Meta Developer.
  2. Sign up and create a Meta Business Manager account.
  3. Create App > Select Business > Name: Galaxy Workers.
  4. Go to WhatsApp > API Setup > Choose Cloud API > Add phone number (+94).
  5. From Configuration, copy Business ID, Phone ID, and Access Token.
